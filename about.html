APLICACIÓN WEB

Aplicacion web simple para gestionar una lista de tareas

Esta es una aplicacion web simple para hacer tareas, esta permite agregar tareas asi mismo poder macarlas como completadas y eliminar una tarea cuando ya ha sido completada

Como primer objetivo aplicar los conocimientos adquiridos sobre HTML, CSS, Bootstrap, Docker, Vercel y control de versiones con Git y GitHub

Y segundo crear una aplicacion web sencilla para gestionar una lista de tareas
